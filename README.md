nghttp2-vagrantfile
===============
nghttp2 build on the ubunt13.10
You can use http2 immediately

use
---------------
boot vm
```
$ vagrant plugin install vagrant-omnibus
$ vagrant up
```

login vm
```
$ vagrant ssh

vagrant@vagrant:~$ ./nghttp2/src/nghttpd
Usage: nghttpd [-DVfhv] [-d <PATH>] [--no-tls] <PORT> [<PRIVATE_KEY> <CERT>]
Too few arguments

```
